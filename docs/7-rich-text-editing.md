# Rich-text editing

Rich-text editing is a **core feature** of Snippets.

The editor is designed to support complex, structured documents while remaining fast, predictable, and easy to use.

## Markdown-like shortcuts (without being Markdown)

Snippets supports familiar shortcuts for creating formatting:

- Type `#` + <kbd>Space</kbd> to create a heading
- Similar patterns exist for lists, quotes, and other block types

## Creating and removing formatting

Formatting is easy to apply and just as easy to remove:

- Use <kbd>Space</kbd> to create formatting
- Use <kbd>Backspace</kbd> to remove formatting

## Navigating formatting boundaries

You can move in and out of formatting using arrow keys:

- <kbd>ArrowLeft</kbd> enters formatting
- <kbd>ArrowRight</kbd> exits formatting

This makes editing precise and predictable.

## Drag & drop structured content

Paragraphs, list items, and blocks are treated as structured nodes.

- Double-click at the end of a block to select it
- Drag and drop it anywhere else in the document

## Built-in toolbar

For more advanced interactions, the editor includes a toolbar that exposes richer formatting and structured elements.

## Designed for complex documents

The editor supports:

- nested blocks
- embedded elements
- references
- mixed structured content

Documents remain robust and extensible as they grow.
