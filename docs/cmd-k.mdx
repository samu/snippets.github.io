# CMD+K – quick access to CLI commands

<kbd>Cmd</kbd> + <kbd>K</kbd> gives you fast access to frequently used CLI
commands.

## How it works

- Store commands inside **code blocks** or **links**
- Press <kbd>Cmd</kbd> + <kbd>K</kbd> to search
- Press <kbd>Enter</kbd> to copy the command to the clipboard

## Why this is useful

- Commands live next to their documentation
- Less context switching
- A personal, searchable command palette

Your notes become an active working tool — not just text.
